# professional-task-manager
Features

Full CRUD Functionality: Create, read, update, and delete tasks dynamically.

Advanced State Sorting: Tasks are automatically double-sorted—first by priority (High > Medium > Low), and secondly, completed tasks are pushed to the bottom to keep the workspace clean.

Data Persistence: Integrated with the browser's LocalStorage API so tasks survive page refreshes.

Dynamic Content Filtering: Real-time filtering for "All", "Pending", and "Completed" tasks,

Mobile-First Responsive Layout:

tech Stack & Concepts Applied

HTML5 & CSS3: Semantic markup, custom CSS variables for theme management, and Flexbox layouts.

Vanilla JavaScript: Event architecture, array manipulation (.sort(), .filter(), .splice()),
